Basic Channel Join and Topic
Connect two clients (A and B) to your server.
In client A: /join #test
In client B: /join #test
In client A: /topic #test This is a test topic
Client B should see the topic change.
In client B: /topic #test
Should display: "This is a test topic".


2. Topic Permissions
In client A (as channel operator): /mode #test +t
In client B (not operator): /topic #test New topic by B
Should receive an error: "You're not channel operator".


3. KICK Command
In client A (as operator): /kick #test <B's_nick>
Client B should be removed from #test and see a kick message.
In client B: /join #test
Should be able to rejoin if not banned.


4. KICK Error Handling
In client B (not operator): /kick #test <A's_nick>
Should receive an error: "You're not channel operator".
In client A: /kick #test not_in_channel
Should receive an error: "They aren't on that channel".


5. Registration and Parameter Errors
Connect a new client C, but do not register (do not send NICK/USER).
In client C: /kick #test <A's_nick>
Should receive an error: "You have not registered".
In client A: /kick
Should receive an error: "Not enough parameters".


6. Miscellaneous
Try /topic #test with no topic set: should get "No topic is set".
Try /kick #notachannel <B's_nick>: should get "No such channel" or "You're not on that channel".


7. JOIN and PART/QUIT
Test: Join a channel

/join #mychan
Should join and see a join message.
Test: Join multiple channels

/join #chan1,#chan2
Should join both.
Test: Part a channel

/part #mychan
Should leave and see a part message.
Test: Quit server

/quit
All joined channels should see a quit message.


8. PRIVMSG and NOTICE
Test: Private message to user

Client A: /privmsg <B's_nick> hello
Client B should receive the message.
Test: Private message to channel

/privmsg #mychan hello
All users in #mychan should receive the message.
Test: Notice

/notice <B's_nick> hi
Client B should receive a notice.


9. INVITE
Test: Invite user to channel

Client A: /invite <B's_nick> #mychan
Client B should receive an invite message.
Test: Invite non-existent user

/invite nothere #mychan
Should get an error: "No such nick/channel".


10. MODE
Test: Set channel operator

Client A: /mode #mychan +o <B's_nick>
Client B should become operator.
Test: Set topic restriction

/mode #mychan +t
Only ops can change topic now.
Test: Remove operator

/mode #mychan -o <B's_nick>
Client B loses operator status.


11. ERRORS AND EDGE CASES
Test: Send command before registration

Connect, but do not send NICK/USER.
Try /join #chan
Should get: "You have not registered".
Test: Join non-existent channel

/join #doesnotexist
Should create and join the channel.
Test: Send message to non-existent user

/privmsg nothere hi
Should get: "No such nick/channel".
Test: Join with invalid channel name

/join invalidchan
Should get: "No such channel" or similar error.
